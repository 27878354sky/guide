# 北京邮电大学 信息与通信工程 2023级 创新实验作业

## 1. 介绍

本仓库为北京邮电大学信息与通信工程2023级创新实验作业，基于Python3.10开发。

## 2. 使用方法

### 2.1 安装依赖

```bash
pip install -r requirements.txt
```

### 2.2 运行

```bash
python main.py
```

### 2.3 Linux 环境配置与运行

在 Linux 环境下，建议按照以下步骤配置虚拟环境并运行：

```bash
# 1. 创建虚拟环境
python3 -m venv venv

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行程序
python main.py
```
